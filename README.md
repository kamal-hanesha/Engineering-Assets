# Engineering-Assets


This repository, Engineering Assets, serves as a curated collection of high-value resources developed from extensive experience securing and deploying mission-critical web applications.

It is primarily focused on secure application deployment and adherence to best practices, with a core asset being the Vulnerability Free Coding Guide. These assets are designed for reliability and security compliance.

Key Assets:

**Vulnerability Free Coding Guide**

The Vulnerability Free Coding Guide (PDF/Markdown) is a comprehensive document detailing all necessary security measures and best practices to be taken care of during the web development lifecycle. This guide is built to help developers create applications that meet VAPT-level readiness and mitigate common security flaws.

**Deployment Automation Script**: 

Scripts and configurations for streamlining the deployment of coupled Django/DRF (Backend) and React JS (Frontend) applications, often utilized on Apache environments with PostgreSQL database.
